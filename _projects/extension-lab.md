## Tóm tắt
Extension này giúp mình copy nhanh nội dung từ Sheet, tạo task và điều hướng API. Tạo bằng HTML/CSS/JS, dùng Chrome Storage để lưu trạng thái.

## Tính năng chính
- Nút “Gửi báo cáo” gọi Apps Script Web App để gửi email summary.
- Lưu tag và trạng thái vào Chrome Storage sync.
- Giao diện đơn giản, tương phản cao, tương tự phong cách Material nhẹ.
